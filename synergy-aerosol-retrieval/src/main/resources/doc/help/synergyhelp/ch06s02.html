6.2. 'Processing Parameters' Tab Pane

Figure 6.2. Aerosol Retrieval and Atmospheric Correction: 'Processing Parameters' Tab Pane


This tab pane consists of the following components:
  • 'Retrieve AODs over ocean': If this checkbox is selected, aerosol optical depths will be retrieved over ocean. The default is 'true'.
  • 'Retrieve AODs over land': If this checkbox is selected, aerosol optical depths will be retrieved over land. The default is 'true'.
  • 'Path to LUTs for land and ocean aerosol algorithms': Text field to set the path to the lookup tables required for the ocean aerosol retrieval. See [2] for more details.
  • 'Soil surface reflectance spectrum': Text field to set the filename containing the soil surface reflectance spectrum. The default is 'spec_soil.dat' and is the only one included in the delivered software package. If the user wants to use different files, they must be added to the software archive (synergy-aerosol-retrieval-1.1-SNAPSHOT.jar) with an appropriate tool before starting Visat.
  • 'Vegetation surface reflectance spectrum': Text field to set the filename containing the vegetation surface reflectance spectrum. The default is 'spec_veg.dat' and is the only one included in the delivered software package. If the user wants to use different files, they must be added to the software archive (synergy-aerosol-retrieval-1.1-SNAPSHOT.jar) with an appropriate tool before starting Visat.
  • 'List of land aerosol models': Text field to set a comma separated list of the indices of the aerosol models to be used for the land aerosol retrieval. The default is '8,20,28'. See [3] for more details.
  • 'N x N average for AOD retrieval': Text field to set the number of pixels to average over in the aerosol retrieval. The default number is '7', which corresponds to a square with 3 pixels to each side of a center pixel.
  • 'Rescale AOD to original resolution after interpolation': If this checkbox is selected, the retrieved AODs retrieved on the coarse N x N grid will be rescaled to the original resolution by a bicubic interpolation.
  • 'Retrieve surface directional reflectances (over land only)': If this checkbox is selected, SDRs will be retrieved from rescaled AODs. This computation is performed over land only and requires AOD rescaling (previous checkbox).
