The Cloud Screening Processor is the second of three main modules within the MERIS/(A)ATSR Synergy toolbox. Its purpose is to add various bands (i.e cloud flags) to the colocated MERIS/(A)ATSR L1b product obtained from the preprocessor. The output product from the cloud screening shall be used as input for the third module (aerosol retrieval and atmospheric correction).

The user interface for the Cloud Screening Processor is represented by one main dialog, which consists of the following components:

  • 'I/O Parameters' Tab Pane
  • 'Processing Parameters' Tab Pane

5.1. I/O Parameters Tab Pane

Figure 5.1. Cloud Screening: I/O Parameters Tab Pane


This tab pane consists of the following components:
  • Source product file: Select the source product file by either typing the product path into the text filed or by pressing the file selection button to invoke a standard file dialog. The source product must be a colocated MERIS/AATSR product generated by the 'Synergy Preprocessor' module of the MERIS/(A)ATSR Synergy Toolbox.
  • Target product file: Select the target product file by either typing the product path into the text filed or by pressing the file selection button to invoke a standard file dialog.
  • Target product format and target directory:
    • 'Save as': If this checkbox is selected, the output product will be saved to disk.
    • Target product format: Select one of the available output product formats.
    • Select the target product directory by either typing the product path into the text filed or by pressing the file selection button to invoke a standard file dialog.
  • 'Open in Visat': If this checkbox is selected, the output product is opened in Visat after successful processing.
