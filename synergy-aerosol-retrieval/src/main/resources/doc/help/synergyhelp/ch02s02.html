2.2. Lookup Tables

To be able to process all options properly, the MERIS/(A)ATSR Synergy Toolbox processor needs the following lookup tables:

  • Ocean aerosol LUTs (provided by FUB)
  • Land aerosol LUTs (provided by USwansea)

Currently, these LUTs are provided by FUB and USwansea on request. Automatic access is in preparation. The LUTs can be deployed anywhere on disk, the path needs to be specified in the User Interface when running the processor.
