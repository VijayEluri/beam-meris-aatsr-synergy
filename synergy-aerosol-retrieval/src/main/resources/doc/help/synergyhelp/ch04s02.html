4.2. 'Processing Parameters' Tab Pane

Figure 4.2. 'Processing Parameters' Tab Pane


This tab pane consists of the following components:
  • 'Retrieve AODs over ocean': If this checkbox is selected, the aerosol optical depths will be retrieved over ocean.
  • 'Path to LUTs for ocean aerosol algorithm': Text field to set the path to the lookup tables required for the ocean aerosol retrieval. See [2] for more details.
  • 'Retrieve AODs over land': If this checkbox is selected, the aerosol optical depths will be retrieved over land.
  • 'Path to LUTs for land aerosol algorithm': Text field to set the path to the lookup tables required for the land aerosol retrieval. See [3] for more details.
  • 'Soil surface reflectance spectrum': Text field to set the filename containing the soil surface reflectance spectrum. The default is 'spec_soil.dat' and is included in the software package.
  • 'Vegetation surface reflectance spectrum': Text field to set the filename containing the vegetation surface reflectance spectrum. The default is 'spec_veg.dat' and is included in the software package.
  • 'List of land aerosol models': Text field to set a comma separated list of the indices of the aerosol models to be used for the land aerosol retrieval. The default is '8,2'. See [3] for more details.
  • 'Pixels to average (n x n, with n odd number) for AOD retrieval': Text field to set the number of pixels to average over in the aerosol retrieval. The default number is '11', which corresponds to a square with 5 pixels to each side of a center pixel.
  • 'Interpolate AOD gaps': If this checkbox is selected, gaps in the retrieved AODs will be interpolated by a Boxcar interpolation.
  • 'Rescale to original resolution after interpolation of AOD gaps': If this checkbox is selected, the retrieved AODs retrieved on the coarse n x n grid will be rescaled to the original resolution by a bicubic interpolation.
  • 'Retrieve SDRs from rescaled AODs (over land only)': If this checkbox is selected, SDRs will be retrieved from rescaled AODs. This computation is performed over land only and requires AOD rescaling (previous checkbox).
